# My second project.
This single-page app was created for users to save products they have tried, rate them, and write descriptions, making it easier for them to keep track of these items. 

### Front-end:  
- HTML
- CSS
- JavaScript
- React
- Redux

### Back-end:  
- Node.js
- Express.js
- PostgreSQL
